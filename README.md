<pre># 📚 Bookstore Management System

A **Bookstore Management System** built using **Python, Flask, MySQL, HTML, CSS, JavaScript, and Bootstrap**. This project provides a simple and user-friendly interface to manage bookstore inventory with complete CRUD operations, search, sorting, filtering, pagination, and form validation.

---

## 🚀 Features

* ➕ Add new books
* 📖 View all books
* ✏️ Edit existing book details
* ❌ Delete books with confirmation
* 🔍 Search books by title or author
* 📂 Filter books by category
* ↕️ Sort books by title, price, or published year
* 📄 Pagination (9 books per page)
* 📊 Dashboard statistics

  * Total Books
  * Total Inventory Quantity
* ✅ Form validation for all input fields
* ⚠️ Custom 404 Error Page
* ⚠️ Custom 500 Error Page
* 🎨 Responsive and clean user interface

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Bookstore-Management-System.git
```

### 2. Navigate to the project folder

```bash
cd Bookstore-Management-System
```

### 3. Create a virtual environment

```bash
python -m venv venv
```

### 4. Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 5. Install dependencies

```bash
pip install -r requirements.txt
```

### 6. Configure the database

Update your database credentials in `config.py`.

### 7. Create the database

```bash
python create_db.py
```

### 8. Run the application

```bash
python app.py
```

### 9. Open your browser

```
http://127.0.0.1:5000
```
